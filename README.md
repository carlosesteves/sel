# sel

## Dependencies

- Java 8
- Gradle
- Firefox 31
- Linux/Mac OS X

## Build

Checkout the code
```
git clone https://github.com/carlosesteves/sel.git
```

Build
```
./gradlew clean compileJava compileTestJava
```

## Run the tests

```
./gradlew test
```

## Reports

The HTML report is located in `/buid/reports/tests/index.html`.
